ricco.util.util.sort_by_list(src_list, by_list, filter_=False) → list

根据一个列表对另一个列表进行筛选或排序，参照列表中不存在的元素按照原始顺序排列在后

参数:

  • src_list – 要进行排序的列表

  • by_list – 参照的列表

  • filter – 是否根据参照列表筛选

示例

>>> a = [1, 2, 3, 4, 5]
>>> b = [2, 5, 4, 1]

[2, 5, 4, 1, 3]
>>> sort_by_list(a, b, filter_=True)
[2, 5, 4, 1]
>>> sort_by_list(b, a)
